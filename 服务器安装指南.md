# OpenMPI的安装
Parthenon框架依赖的MPI技术本质上和C++标准一样只是一种协议，各个厂商有自己的实现方式。这里我们介绍OpenMPI的安装方法。首先在你的主目录从官网下载并解压OpenMPI源码：
```sh
wget https://download.open-mpi.org/release/open-mpi/v5.0/openmpi-5.0.10.tar.gz
tar -zxvf openmpi-5.0.10.tar.gz
```
如此一来，主目录下应当出现一个名为`openmpi-5.0.10`的文件夹。进入这个文件夹并按照如下方式进行配置：
```sh
cd openmpi-5.0.10
./configure --prefix=/openmpi --with-cuda=/usr/lib/cuda --enable-mpi-cxx --enable-shared
```
下一步，使用make工具进行编译并构建：
```sh
make all install
```
顺利安装好后，我们改写环境变量并加入openmpi的安装路径：
```sh
vim ~/.bashrc
```
输入：
```sh
export PATH=$PATH:/openmpi/bin
export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/openmpi/lib
```
保存后使用`source ~/.bashrc`应用新的配置。

# Parallel HDF5的安装
回到主目录，下载并解压HDF5 1.14的源码压缩包：
```sh
wget https://support.hdfgroup.org/releases/hdf5/v1_14/v1_14_3/downloads/hdf5-1.14.3.tar.gz
tar -zxvf hdf5-1.14.3.tar.gz
```
进入到解压文件夹：
```sh
cd hdfsrc
```
随后设置Parallel HDF5的安装选项：
```sh
export CC=mpicc
export HDF5_MPI="ON"
./configure --enable-shared --enable-parallel --prefix=/hdf5
```
顺利配置后，设置HDF5安装文件夹：
```sh
export HDF5_DIR="/hdf5"
```
之后，输入`make`启动编译。
```sh
make install
make check-install
export PATH=$PATH:"/hdf5/bin"
export HDF5_ROOT="/hdf5"
```
之后使用`source ~/.bashrc`应用这一设置。
